# Toolchains

This repository contains a pre-compiled version of the LLVM for building The Thrush Compiler and using LLVM backend in between.

## LLVM 

#### Linux

- VERSION: **v17.0.6** 
- OPERATING SYSTEMS: Linux (x86_64)

#### Windows

- VERSION: **v18.0.8** 
- OPERATING SYSTEMS: Windows (x86_64)

### LLVM - HOST TARGETS

Host targets mean that LLVM was precompiled specifically for that target which will use the Thrush compiler to compile itself.

#### Windows

- `x86_64-pc-windows-msvc`
- `x86_64-pc-windows-gnu`

### Linux

- `x86_64-unknown-linux-gnu`
- `x86_64-unknown-linux-musl`

### LLVM - AVAILABLE TARGETS

- ``x86_64``

- ``RISC-V``

- ``MIPS``

- ``NVPTX``
- ``SPARC``
